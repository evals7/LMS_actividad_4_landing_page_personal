# Landing-Page-Personal
🧾 Landing Page - Eva López Sainz-Aja
📌 Descripción
Este proyecto consiste en una landing page personal desarrollada utilizando HTML5 y CSS3 para la asignatura Lenguaje de Marcas del CFGS DAM, con un diseño moderno y adaptable gracias al uso de Flexbox y técnicas de responsive design. La página actúa como una presentación profesional, incluyendo acceso al CV, proyectos, formulario de contacto y redes sociales.

🛠️ Tecnologías utilizadas
    HTML5 – Estructura semántica de la página.
    CSS3 – Estilos personalizados.
    Flexbox – Organización del layout en bloques flexibles.
    Responsive Design – Adaptación a distintos tamaños de pantalla (desktop, tablet, móvil).
    Font Awesome – Uso de iconos visuales para navegación y redes sociales.

🎯 Funcionalidades
    Menú de navegación responsive con ícono burguer en dispositivos móviles.
    Foto de perfil y breve descripción profesional.
    Enlaces a redes sociales (LinkedIn, GitHub, Instagram).
    Enlaces internos a otras secciones: CV, Proyectos y Contacto.
    Uso de @media queries para adaptar el contenido a diferentes resoluciones.

📱 Responsive design
La landing page está diseñada para adaptarse correctamente a dispositivos de distintos tamaños:
    En pantallas grandes, el contenido se distribuye horizontalmente (foto a un lado, texto al otro).
    En pantallas pequeñas, el layout se reorganiza verticalmente para mejorar la lectura y usabilidad.
    El menú se transforma en un botón de navegación (hamburguesa) usando checkbox y label como disparadores.
